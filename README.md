# RoTan

Endpoint:<br>

**Lahan:**<br>
`Lahan.getSemua()` (static)<br>
Mengembalikan ArrayList semua lahan yang dimiliki oleh pengguna.<br><br>
`Lahan.hapus(int)` (static)<br>
Menghapus data lahan berdasarkan kode lahan (tersimpan).<br><br>
`Lahan.insert(Lahan)` (static)<br>
Menambahkan lahan baru (tersimpan).<br><br>

**Aktivitas:**<br>
`Aktivitas.getSemua()` (static)<br>
Mengembalikan ArrayList semua aktivitas dari semua lahan yang dimiliki oleh pengguna.<br><br>
`Aktivitas.getAktivitasLahan(int)` (static)<br>
Mengembalikan ArrayList semua aktivitas dari lahan tertentu berdasarkan kode lahan.<br><br>
`Aktivitas.insert(Aktivitas)` (static)<br>
Menambahkan aktivitas baru (tersimpan).<br><br>
